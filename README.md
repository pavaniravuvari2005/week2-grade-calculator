# Student Grade Calculator

## Project Description
A Python program that calculates student grades using control flow,
lists, loops, and error handling.

## What I Learned
- Conditional statements (if/elif/else)
- Lists and loops
- Input validation
- Functions and modular coding
- Error handling using try-except

## Features
- Multiple student processing
- Grade calculation
- Personalized comments
- Class statistics
- Input validation

Grading System

A: 90–100

B: 80–89

C: 70–79

D: 60–69

F: Below 60
